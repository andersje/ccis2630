• Creating a session: initialize session
  • To create a session variable, simply use the $_SESSION[ ] array
    •  session_start();//initializes session
$_SESSION['myVar']="Session Variable value";
  • To access a variable:
    •  session_start();//initializes session
$var = $_SESSION['myVar'];
echo $var;
  • Howabout using arrays? Sure:
    •  Page 1:
session_start();//initializes session
$var = array(1=>"hello",2=>"world");
$_SESSION['var1']=$var; 
     Page 2:
session_start(); //initializes session
foreach($_SESSION AS $key=>$val) {
  foreach($val AS $k=>$v) {
    print "KEY:$k -- VALUE:$v<br />";
  }
}
