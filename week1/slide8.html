• Create layouts using CSS not tables
  • If applets and plug-ins are required, make sure alternative text versions are available
  • Create forms making use of legend, label, and fieldset elements
  • Make sure all labels relate to the correct element
  • Always use the alt attribute on images
  • Use the title attribute on other elements when needed
  • links/lynx are good tests for how 508-compliant your page is
  • avoid frames--you don't need them
