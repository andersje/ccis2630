• All pages written in this class must be XHTML 1.0 compliant
  • All documents start with a valid DTD
  • All documents are contained with <html> </html> tags
  • All <html> tags contain both a <head> </head> and a <body> </body> section.
  • <head> sections: titles, meta data, javascript, css info
  • <body> sections contain all of the content
  • Validator at http://validator.w3.org/
  • Remember: All tags must be closed, even simple line break and horizontal rules: < br />
